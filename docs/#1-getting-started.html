Getting Started with Your C++ Project

Welcome to your C++ project. This document will guide you through the initial setup and help you get started quickly.

Prerequisites

Before you begin, ensure you have the following installed on your machine:

  • C++ Compiler: A C++11-compatible compiler (e.g., g++, clang++)
  • CMake: For building the project
  • OpenGL: Version 3.3 or higher
  • GLFW: For window and input management
  • GLM: For vector math (included in the project)
  • GLAD: For OpenGL function loading (included in the project)

Installing from Source

Follow these steps to set up your project:

    Linux

  1. Installing for a Specific Project:

    Clone the Gitea Repository 
    git clone https://git.openspm.org/OPENSPM/SGL.git
    or Clone from the GitHub Repository 
    git clone https://github.com/ghillie575/sgl.git

    Navigate into the project directory:

    cd SGL
    Build the Project 
    mkdir build
    cd build
    cmake ..
    cmake --build . --config Release
    cmake --install . --prefix /path/to/project

    lib and include directories will be created, or if already created lib and include files will be put in these directories

    here is example project structure you will get after installation:

    .
                ├── include
                │   ├── sgl
                │   │   ├── dynamicstore.h
                │   │   ├── logger.h
                │   │   ├── object.h
                │   │   ├── scenedata.h
                │   │   ├── scenemanager.h
                │   │   ├── sceneobject.h
                │   │   ├── shader.h
                │   │   ├── time_utils.h
                │   │   ├── transform.h
                │   │   └── window.h
                │   └── your_include_files.h
                ├── lib
                │   ├── libsgl.so
                │   └── your_libs.so
                └── your_cpp_file.cpp

  2. Installing System-Wide:

    Clone the Gitea Repository 
    git clone https://git.openspm.org/OPENSPM/SGL.git
    or Clone from the GitHub Repository 
    git clone https://github.com/ghillie575/sgl.git

    Navigate into the project directory:

    cd SGL
    Build the Project 
    mkdir build
    cd build
    cmake ..
    cmake --build . --config Release
    cmake --install . --prefix /usr
    (must be run as root)

Running the Project

To start the test scene, run the following command from the build directory:

./XandO
